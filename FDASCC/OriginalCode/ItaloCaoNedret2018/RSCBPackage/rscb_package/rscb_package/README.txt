M-Based Simultaneous Inference for the Mean Function of Functional Data

Italo R. Lima, Guanqun Cao, Nedret Billor


------------

/\/\/\/\/\/\/
INSTALLATION:
\/\/\/\/\/\/\




1) Make sure that the following libraries are installed in your system:
- splines
- MASS
- reshape2



2) Install package using R command:

> install.packages(‘/dir/topackage/RSCB_0.1.0.tgz’)



3) Load package using R command:

> library(‘RSCB’)





/\/\/\/\/\
Functions:
\/\/\/\/\/



mRSCB - Calculates the robust estimation for the mean function and robust simultaneous confidence band. See ?mRSCB for more details; 



mRSCB_twosample - Calculates the robust estimation and robust simultaneous confidence band for the difference of the mean functions. See ?mRSCB_twosample for more details;




/\/\/\/\
Example:
\/\/\/\/



run sample_RSCB.R;
