# FDASCC Package in Python with rpy2 Wrapper
--------------------------------------------
### To run the exmaples in example.R:
1. Install R
2. Install gtools, splines2, pracma, RSpectra libraries using install.packages
3. changed the function combinations in cov1g.sce.R and cov2g.sce.R to combn
4. install.packages(<path to the FDASCC package written in R>, repos = NULL, type="source")
