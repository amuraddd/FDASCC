# FDASCC Package in Python with rpy2 Wrapper
--------------------------------------------
### To run the exmaples in example.R:
1. Install R

The following steps only need to be done once to set up the base R environment:  
2. Install gtools, splines2, pracma, RSpectra libraries using install.packages
3. install.packages("path to the FDASCC package written in R", repos = NULL, type="source")
